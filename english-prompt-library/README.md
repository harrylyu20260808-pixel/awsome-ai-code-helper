## 📖 Overview

A comprehensive, bilingual prompt template library designed for AI coding assistants. This application provides ready-to-use templates for Python programming beginners, covering all essential concepts from basic functions to advanced algorithms.

这是一个全面的、双语AI编程助手提示词模板库，专为Python编程初学者设计，涵盖从基础函数到高级算法的所有核心概念。

## ✨ Features

- 📚 **60+ Pre-built Templates**: Comprehensive collection of programming templates / **60+ 预构建模板**：全面的编程模板集合
- 🌐 **Bilingual Support**: English and Chinese templates / **双语支持**：英文和中文模板
- ✏️ **Custom Templates**: Create, edit, and manage your own prompts / **自定义模板**：创建、编辑和管理自己的提示词
- 💾 **Persistent Storage**: Templates saved in browser localStorage / **持久化存储**：模板保存在浏览器localStorage中
- 📋 **One-Click Copy**: Easily copy prompts to clipboard / **一键复制**：轻松复制提示词到剪贴板
- 🎨 **Beautiful UI**: Modern, responsive design with animations / **美观界面**：现代、响应式设计配合动画效果
- 🔄 **Easy Navigation**: Intuitive sidebar navigation / **简单导航**：直观的侧边栏导航

## 🎯 How to Use

### Using Built-in Templates

1. Click on a template category in the sidebar / 在侧边栏点击模板分类
2. Browse the available prompts / 浏览可用的提示词
3. Click "📋 Copy Prompt" button to copy / 点击"📋 Copy Prompt"按钮复制
4. Paste the prompt into your AI coding assistant / 将提示词粘贴到AI编程助手中

### Using Custom Templates

1. Navigate to "✏️ Custom Templates" in the sidebar / 在侧边栏导航到"✏️ Custom Templates"
2. Click "Create Template" button / 点击"Create Template"按钮
3. Fill in the template information / 填写模板信息：
   - **Title** (Required) / **Title**（必填）
   - **Description** (Optional) / **Description**（可选）
   - **Content** (Required) / **Content**（必填）
4. Click "Save Template" to save / 点击"Save Template"保存
5. Your template will appear in the list and be available for copying / 模板将出现在列表中并可供复制

### Customization

To add new templates:

1. Open `app.js`
2. Find the template category you want to add to
3. Add a new template object in the array
4. Example:
```javascript
functions: [
    {
        title: 'Your Template Title / 你的模板标题',
        description: 'Description in English / 中文描述',
        content: `Your template content here with placeholders. / 你的模板内容，包含占位符。`
    }
]
```

### Adding Custom Templates

To add custom templates programmatically:

```javascript
const newTemplate = {
    id: Date.now().toString(),
    title: 'My Custom Template',
    description: 'This is my custom template',
    content: 'Your custom prompt content...',
    createdAt: new Date().toISOString()
};

saveTemplateToStorage(newTemplate);
```
