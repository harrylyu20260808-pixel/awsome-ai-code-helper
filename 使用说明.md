# 🤖 AI Code Helper - 编程新手助手

面向编程新手的AI编程助手，让新手无需写prompt，也无需懂调试，就能生成并运行代码。

## ✨ 核心功能

### 1. 表单式代码生成
- 用户通过简单表单输入需求
- 系统自动生成高质量prompt
- 调用AI生成代码
- 返回可运行的代码文件

### 2. 自动调试功能
- 用户粘贴报错信息
- 系统自动分析错误原因
- 自动生成修复代码
- 返回修复后的代码

### 3. 逐行解释代码
- 对生成的代码进行逐行解释
- 用新手能理解的语言说明每一行的作用
- 支持任意行号的解释

### 4. 提示词模板库
- 包含60+个预构建的编程提示词模板
- 支持英语和中文内容
- 可自定义创建新模板
- 持久化存储到浏览器localStorage

## 🚀 快速开始

### 环境要求
- Python 3.8+
- pip包管理器
- 现代浏览器（支持 ES6+）

### 安装步骤

#### 1. 安装依赖

**Linux/Mac:**
```bash

pip3 install -r requirements.txt
```

**Windows:**
```bash
cd C:\path\to\ai-code-helper
pip install -r requirements.txt
```

#### 2. 配置AI API（重要！）

编辑 `app.py` 文件，找到 `call_ai()` 函数，接入你的AI API。

**选项A: 使用Mock模式（无需API密钥）**

直接使用内置的mock响应进行测试：
```python
# app.py - call_ai 函数
async def call_ai(prompt: str) -> str:
    # 简单的mock实现
    language = "Python"
    if "JavaScript" in prompt or "javascript" in prompt:
        language = "JavaScript"
    elif "Java" in prompt:
        language = "Java"
    elif "C++" in prompt:
        language = "C++"

    return f"""# Generated Code ({language})
# This is a mock response for testing purposes

def example_function():
    print("Hello from {language}!")

if __name__ == "__main__":
    example_function()
"""
```

**选项B: 接入Claude API**

1. 安装依赖：
```bash
pip install anthropic
```

2. 获取API密钥：https://console.anthropic.com/

3. 在 `app.py` 中配置：
```python
import anthropic
import os

async def call_ai(prompt: str) -> str:
    client = anthropic.Anthropic(api_key=os.environ.get("ANTHROPIC_API_KEY"))
    response = client.messages.create(
        model="claude-3-5-sonnet-20241022",
        max_tokens=2048,
        messages=[{"role": "user", "content": prompt}]
    )
    return response.content[0].text
```

4. 设置环境变量：
```bash
export ANTHROPIC_API_KEY="your-api-key-here"
```

**选项C: 接入OpenAI API**

1. 安装依赖：
```bash
pip install openai
```

2. 获取API密钥：https://platform.openai.com/

3. 在 `app.py` 中配置：
```python
import openai
import os

async def call_ai(prompt: str) -> str:
    client = openai.AsyncOpenAI(api_key=os.environ.get("OPENAI_API_KEY"))
    response = await client.chat.completions.create(
        model="gpt-3.5-turbo",
        messages=[
            {"role": "system", "content": "You are a coding expert. Generate code only."},
            {"role": "user", "content": prompt}
        ],
        max_tokens=2048
    )
    return response.choices[0].message.content
```

## 📖 使用指南

### 场景1：代码生成

#### 步骤1：输入需求
在"代码生成"标签页中：
- 在"你的需求"框中输入：`写一个Python函数，求两个数的和`
- 选择编程语言：`Python`

#### 步骤2：查看结果
系统会：
1. 自动生成高质量的prompt
2. 调用AI生成代码
3. 在页面下方显示生成的代码

#### 步骤3：查看解释
点击"查看解释"按钮，可以逐行了解代码的作用。

#### 步骤4：复制或下载
- 点击"复制代码"：代码会自动复制到剪贴板

### 场景2：自动调试

#### 步骤1：粘贴报错信息

- 在"出错代码"框中粘贴你的代码

#### 步骤3：点击"自动修复"
系统会：
1. 分析错误原因
2. 调用AI生成修复后的代码
3. 在页面下方显示修复后的代码

#### 步骤4：查看结果
- 错误分析：系统会解释错误类型和原因
- 修复后的代码：AI自动修复的问题代码
- 复制或下载：可以复制或下载修复后的代码

### 场景3：代码解释

#### 步骤1：输入代码
切换到"代码逐行解释"页面：
- 在"你的代码"框中粘贴要解释的代码

#### 步骤2：点击"开始解释"
系统会：
1. 逐行分析代码
2. 为每一行生成通俗易懂的解释
3. 用emoji图标和清晰的格式显示

#### 步骤3：查看解释
- 每一行都有详细的说明
- 标注了每个关键词的作用
- 适合编程新手理解

## 🐛 常见问题

### Q: 代码生成失败怎么办？
A: 检查以下事项：
1. AI API配置是否正确
2. 网络连接是否正常
3. API密钥是否有效
4. Prompt是否合理

### Q: 调试功能不工作？
A: 检查：
1. 错误信息是否完整
2. 代码是否粘贴正确
3. API调用是否成功

### Q: 如何部署到生产环境？
A: 推荐使用：
- Uvicorn + Nginx
- Gunicorn + Gevent
- Docker容器化

### Q: 提示词库无法加载自定义模板？
A: 确保浏览器支持localStorage，并检查浏览器控制台是否有错误。
